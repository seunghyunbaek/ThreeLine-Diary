# ThreeLineDiary(세줄일기)
19.04 ~ 19.06

### 소개
세줄일기라는 애플리케이션은 스토어에 올라와있는 애플리케이션입니다. 출시되어있는 애플리케이션을 가이드라인삼아 만들어나갔습니다.
